Согласно предоставленным источникам, ниже приведен программный код для настройки базы данных и первичного наполнения данными (seeds), соответствующий этапам разработки проекта.
1. Настройка подключения к БД (config/database.yml)
Для работы приложения используется СУБД PostgreSQL 1, 2. В файле конфигурации определяются параметры подключения для разных сред (development, test, production), включая имя пользователя, хост и ссылки на секретные ключи 1.
default: &default
  host: localhost
  adapter: postgresql
  encoding: UTF8
  username: sinatra_admin # Имя созданной роли в PostgreSQL
  timeout: 5000
  reaping_frequency: 30
  pool: <%= ENV.fetch("RAILS_MAX_THREADS") { 5 } %>


development:
  <<: *default
  database: api_hirer_development
  password: <%= Rails.application.secrets[:database][:password] %>


test:
  <<: *default
  database: api_hirer_test
  password: <%= Rails.application.secrets[:database][:password] %>


production:
  <<: *default
  database: api_hirer_production
  password: <%= ENV['EXERCISE_DATABASE_PASSWORD'] %>
1
2. Секретные ключи (config/secrets.yml)
Этот файл используется для хранения паролей БД и ключа secret_key_base 1. Важно: данный файл рекомендуется добавлять в .gitignore для безопасности 1.
development:
  database:
    :password: password # Пароль роли PostgreSQL
  secret_key_base: 52e1ae7712b76d320208.... # Генерируется через rake secret


test:
  database:
    :password: password
  secret_key_base: b5db85fb51a88...


production:
  database:
    :password: <%= ENV["PRODUCTION_DATABASE_PASSWORD"] %>
  secret_key_base: <%= ENV["SECRET_KEY_BASE"] %>
1
3. Наполнение данными (db/seeds.rb)
В файле seeds.rb реализована логика очистки старых данных, сброса счетчиков ID и создания новых записей для пользователей, тем и изображений 2.
Создание изображений и тем:
# Очистка и сброс последовательностей
Image.delete_all
Image.reset_pk_sequence


# Создание массива изображений
Image.create([
  {name: 'P.-A. Renoir, Бал в Мулен де ла Галетт', file: 'Renoir Бал в Мулен де ла Галлет.jpg', theme_id: 2},

4. Вспомогательные настройки (Initializers)
Чтобы Rails корректно подгружал изображения из вашей папки, в файл config/initializers/assets.rb необходимо добавить путь к папке с картинками 2:
Rails.application.config.assets.paths << Rails.root.join("assets", "images", "pictures")
